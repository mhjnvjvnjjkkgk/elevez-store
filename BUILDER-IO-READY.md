### "Can't see my website in the editor"
**Solution:**
- Make sure your dev server is running: `npm run dev`
- Use the correct URL: `http://localhost:5173/`
- Check that your website loads in a normal browser first

### "Changes don't appear on my site"
**Solution:**
- Make sure you clicked "Publish" in Builder.io
- Clear your browser cache (Ctrl+Shift+R)
- Wait 30 seconds for CDN to update

### "API key error"
**Solution:**
- Your API key is already configured: `273eceb4203548428b50f961521eccd0`
