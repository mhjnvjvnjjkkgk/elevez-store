# ELEVEZ E-Commerce Store

## Quick Start

### 1. Start the Application

Double-click: **START-SIMPLE.bat**

This starts:
- Main website on http://localhost:5173
- Admin API server on port 3001

### 2. Access Your Dashboard

Open: **http://localhost:5173**

### 3. Use Sync & Deploy

In your admin section:
1. Make changes (add/edit products, collections, etc.)
2. Click "Sync & Deploy" button
3. Changes go live in 2 minutes âœ¨

## Tech Stack

- React + TypeScript + Vite
- Firebase (Database & Auth)
- Tailwind CSS
- Deployed on Vercel

## Deployment

The "Sync & Deploy" button automatically:
1. Commits changes to GitHub
2. Triggers Vercel deployment
3. Updates live site

## Ports

- **5173** - Main dashboard (use this)
- **3001** - Admin API (runs in background)

## Support

For issues, check the console logs or restart both servers using START-SIMPLE.bat
