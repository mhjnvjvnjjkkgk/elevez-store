üîß Troubleshooting

Index stuck on "Building":

  • This is normal, just wait
  • Can take up to 10 minutes for first index
  • Don't refresh the page

Can't find "Create Index" button:

  • Make sure you're in Firestore Database (not Realtime Database)
  • Click the "Indexes" tab at the top
  • You need to be project owner

Still getting errors:

  • Check all 3 indexes are "Enabled"
  • Clear browser cache
  • Wait a few more minutes
  • Try signing out and back in
