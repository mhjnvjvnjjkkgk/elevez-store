# Elevez Redesign: Stitch Prompts

Use these prompts with Stitch (or your preferred UI generation tool) to create stunning, modern Neo-Brutalist components for the Elevez platform.

## Design Philosophy: Modern Neo-Brutalist
*   **Bold Typography**: Massive, high-contrast headings (Syne, Outfit, or Space Grotesk).
*   **Harsh Borders**: Solid 4px-8px black borders.
*   **Offset Shadows**: Large, non-blurred shadows (typically 8px-16px offset).
*   **Vibrant Accents**: Use Neon Green (#00ff88), Cyan (#00ffff), or Magenta (#ff00ff) sparingly.
*   **Clean Layouts**: Plenty of whitespace, grid-based alignment.
*   **Micro-interactions**: Subtle hover reveals, liquid filters, or glitch effects.

---

## 1. Hero Section: "The Cinematic Entry"
**Prompt:**
> Create a high-impact Hero section for a premium streetwear brand named "ELEVEZ". Use a clean white background with a massive black 8px border container. The main headline should be "ELEVATE YOUR STYLE GAME" in 10vw font size, using Syne Black. Split the headline: "ELEVATE YOUR" in black, "STYLE GAME" in Neon Green with a black outline. Add a 3px black horizontal divider below the headline. Include two large, chunky buttons with 8px offset shadows. Add a subtle noise overlay and a scrolling marquee at the bottom with "NEW DROPS // LIMITED EDITION". Ensure the layout is responsive and feels cinematic.

## 2. Best Sellers: "The Product Grid"
**Prompt:**
> Design a modern Neo-Brutalist product grid. Each product card should have a 4px black border and an 8px offset shadow that turns Neon Green on hover. Images should be grayscale and turn to color with a liquid distortion filter on hover. Product names should be in bold uppercase black text. Prices should be large and prominent. Include a "Quick View" button that appears on hover. Use a clean, structured grid with equal spacing.

## 3. Benefits / Why Choose Us: "The Protocol"
**Prompt:**
> Create a "Why Choose Us" section titled "THE PROTOCOL". Use four cards arranged in a grid. Each card features a large Lucide icon in a Neon Green box with a black border. Title the benefits: "3-Day Express", "180gsm Premium", "Excellent Designs", and "Personality-Driven". Use a minimalist aesthetic with bold black text and a subtle 2px grid background. Each card should have a small "Status: Optimized" tag in the corner.

## 4. Recommended Section: "Curated Selection"
**Prompt:**
> Design a "Recommended for You" carousel. Use clean, rounded-corner cards (16px radius) but maintain the Neo-Brutalist 4px black border. Ensure product names are in high-contrast black text. Use a soft gradient background for the section to distinguish it from the main content. Add a "View All Archives" button at the bottom with a magnetic hover effect.

## 5. Newsletter: "The Syndicate"
**Prompt:**
> Create a bold newsletter signup section called "JOIN THE SYNDICATE". Use a solid black background with Neon Green text. The input field should have a 4px white border and no background. The submit button should be a massive Neon Green block with black text. Add a glitch effect to the title. Include a disclaimer in small, high-contrast uppercase text: "DATA ENCRYPTED. NO SPAM PROTOCOLS."

## Technical Integration Tips
*   **Framer Motion**: Use `stiffness: 400, damping: 30` for that "snappy" Neo-Brutalist feel.
*   **Colors**: Primary: `#FFFFFF`, Secondary: `#000000`, Accent: `#00ff88`.
*   **Shadows**: `box-shadow: 8px 8px 0px 0px #000000;`
