# Plan 7.1: Symmetrical Hero Branding & Cursor-Reactive Models

## Objective
Transform the Hero landing page into a stunning, symmetrical, high-fashion storefront. This plan aligns the typography symmetrically, introduces two premium, transparent high-fashion model PNG placeholders flanking the text block, and programs them to glide, tilt, and react dynamically to mouse cursor movements.

- .gsd/SPEC.md
- App.tsx
