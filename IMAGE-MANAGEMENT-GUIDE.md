## ğŸ› Troubleshooting

### **Bulk import still adding empty images?**

**Check:**
1. Are you pasting valid BBCode?
2. Do URLs start with http:// or https://?
3. Check browser console for errors

**Solution:**
- Use direct image URLs
- Copy BBCode correctly from PostImage
- Try single URL first to test

---

### **Can't select images?**

**Check:**
1. Are you clicking on the image itself?
2. Not clicking on action buttons?

**Solution:**
- Click on image area (not buttons)
- Look for âœ“ badge to confirm selection
- Try clicking center of image

---

### **Undo not working?**

**Check:**
1. Is undo button enabled?
2. Have you made any changes?
3. Did you close and reopen modal?

**Solution:**
- Undo only works for current session
- History cleared when modal closed
- Make sure you have changes to undo

---

### **Keyboard shortcuts not working?**

**Check:**
1. Is modal/form focused?
2. Are you in an input field?

**Solution:**
- Click outside input fields
- Focus on the image grid
- Try clicking undo button instead
