==============================================

Connect your project to GitHub for automatic deployments to Vercel.

STEP 1: INSTALL GIT
------------------
1. Download Git for Windows:
   - Go to https://git-scm.com/download/win
   - Download the Windows installer
   - Run the installer with default settings

STEP 2: INITIALIZE GIT IN YOUR PROJECT
------------------------------------
1. Open PowerShell in your project directory:
   cd "d:\wbeiste\elevez (1)"

2. Initialize Git repository:
   git init

3. Add all files:
   git add .

4. Make initial commit:
   git commit -m "Initial commit"

STEP 3: CREATE GITHUB REPOSITORY
------------------------------
1. Go to https://github.com/
2. Sign up or log in to your account
3. Click the "+" icon in the top right
4. Select "New repository"
5. Name it "elevez-store" (or any name you prefer)
6. Keep it Public (not Private)
7. Do NOT initialize with a README
8. Click "Create repository"

STEP 4: CONNECT LOCAL REPOSITORY TO GITHUB
----------------------------------------
1. In PowerShell, add the remote origin:
   git remote add origin https://github.com/YOUR_USERNAME/elevez-store.git
   (Replace YOUR_USERNAME with your actual GitHub username)

2. Push your code:
   git push -u origin main

STEP 5: CONNECT TO VERCEL
------------------------
1. Go to https://vercel.com/
2. Sign up or log in with your GitHub account
3. Click "New Project"
4. Find and select your "elevez-store" repository
5. Vercel will automatically detect it's a Vite project
6. Click "Deploy"

AUTOMATIC DEPLOYMENTS
--------------------
After setup, every time you:
1. Make changes to your code
2. Commit the changes: git commit -m "Description of changes"
3. Push to GitHub: git push origin main

Vercel will automatically:
1. Detect the new code
2. Build your project
3. Deploy the updated site

This means you'll NEVER have to manually deploy again!

BENEFITS:
- Automatic deployments on every push
- Preview deployments for pull requests
- Rollback to previous versions
- Performance monitoring
- Global CDN distribution
